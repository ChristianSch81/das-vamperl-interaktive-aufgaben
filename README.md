# Das Vamperl – Interaktive Aufgaben

Ein interaktives Lernpaket für **Deutsch Phase 3** zum Kinderbuch „Das Vamperl“ von Renate Welsh.

## Inhalt

- interaktive Aufgaben zu elf Kapiteln
- Abschlussquiz mit unmittelbarem Feedback
- kreisförmige Lernfortschrittsanzeige und Lernsterne
- automatische lokale Speicherung
- Import und Export des Lernstands als JSON-Datei
- responsive Darstellung für Computer, Tablet und Smartphone
- eigenständig erzeugte, urheberrechtlich unbedenkliche Figurenillustration

## Verwendung

Die Datei `Das-Vamperl-Interaktiv.html` kann ohne Installation direkt in einem modernen Browser geöffnet werden. Damit die Illustration angezeigt wird, muss `vamperl-figuren-v2.png` im selben Ordner liegen.

Die Lernstände werden zunächst im lokalen Browserspeicher abgelegt. Über die Fortschrittsseite können sie zusätzlich exportiert und später wieder importiert werden.
